# AI Slop Detector

Welcome to the AI Slop Detector documentation. This enterprise-grade suite provides a comprehensive machine learning system designed to identify AI-generated text ("slop") with high confidence and minimal latency.

## Overview

The AI Slop Detector utilizes a state-of-the-art sequence classification transformer model, aggressively optimized via LoRA (Low-Rank Adaptation) and fp16 precision. It provides two methods of interaction:
1. **REST API**: For integrating into existing pipelines and services.
2. **Command Line Interface (CLI)**: For local testing, training orchestration, and manual evaluation.
